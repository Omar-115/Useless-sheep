## 💳 ربط Stripe للدفع الحقيقي

### الخطوات:

#### 1. أنشئ حساب Stripe مجاني:
```
https://stripe.com
```
- انقر "Get started"
- أكمل التسجيل البريدي

#### 2. احصل على المفاتيح (API Keys):
```
Dashboard → Developers → API Keys
```
ستجد:
- **Publishable Key** (عام - استخدمه في HTML)
- **Secret Key** (سري - لا تشاركه مع أحد!)

#### 3. أضف Stripe.js إلى الموقع:

أضف هذا السطر في `<head>`:
```html
<script src="https://js.stripe.com/v3/"></script>
```

#### 4. استبدل الدالة `checkout()`:

ابحث عن هذه الدالة (السطر ~275):
```javascript
function checkout() {
    if (cart.length === 0) return;
    
    const { total } = calculateTotal();
```

استبدلها بـ:
```javascript
function checkout() {
    if (cart.length === 0) return;
    
    const { total } = calculateTotal();
    const stripe = Stripe('YOUR_PUBLISHABLE_KEY');
    
    // أرسل البيانات إلى خادمك
    fetch('/create-payment-intent', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            total: Math.round(total * 100),
            items: cart
        })
    })
    .then(res => res.json())
    .then(data => {
        // فتح نافذة الدفع
        return stripe.redirectToCheckout({ sessionId: data.sessionId });
    });
}
```

**ملاحظة**: تحتاج إلى خادم (Backend) بسيط - راجع الخطوة التالية

## 🖥️ إعداد الخادم (Backend) - الخطوة الأهم

### الخيار 1: استخدام Node.js + Express (الأفضل للمبتدئين)

#### أ) ثبت Node.js:
```
https://nodejs.org/
```

#### ب) أنشئ مجلد جديد:
```bash
mkdir my-store
cd my-store
```

#### ج) ثبت Stripe:
```bash
npm install stripe express cors
```

#### د) أنشئ ملف `server.js`:
```javascript
const express = require('express');
const stripe = require('stripe')('YOUR_SECRET_KEY');
const cors = require('cors');

const app = express();
app.use(cors());
app.use(express.json());

app.post('/create-payment-intent', async (req, res) => {
    try {
        const session = await stripe.checkout.sessions.create({
            payment_method_types: ['card'],
            line_items: req.body.items.map(item => ({
                price_data: {
                    currency: 'usd',
                    product_data: { name: item.name },
                    unit_amount: Math.round(item.price * 100),
                },
                quantity: item.quantity,
            })),
            mode: 'payment',
            success_url: 'YOUR_DOMAIN/success.html',
            cancel_url: 'YOUR_DOMAIN/cancel.html',
        });
        
        res.json({ sessionId: session.id });
    } catch (error) {
        res.status(400).json({ error: error.message });
    }
});

app.listen(3000, () => {
    console.log('Server running on http://localhost:3000');
});
```

#### هـ) شغّل الخادم:
```bash
node server.js
```

### الخيار 2: استخدام Firebase (لا خادم مطلوب!)

**الأسهل للمبتدئين!**

1. اذهب إلى [firebase.google.com](https://firebase.google.com)
2. أنشئ مشروع جديد
3. استخدم Firestore للتخزين
4. أضف هذا الكود في HTML:

```javascript
import { initializeApp } from 'https://www.gstatic.com/firebasejs/9.0.0/firebase-app.js';
import { getFirestore, collection, addDoc } from 'https://www.gstatic.com/firebasejs/9.0.0/firebase-firestore.js';

const firebaseConfig = {
    apiKey: "YOUR_API_KEY",
    projectId: "YOUR_PROJECT_ID",
    // ... الإعدادات الأخرى
};

const app = initializeApp(firebaseConfig);
const db = getFirestore(app);

async function checkout() {
    await addDoc(collection(db, 'orders'), {
        items: cart,
        total: calculateTotal().total,
        timestamp: new Date()
    });
    alert('Order saved! ✅');
}
```
